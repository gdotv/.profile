# gdotv — Graph Database IDE

**gdotv** is a multi-dialect graph database IDE and client for querying, visualizing, debugging, and exploring connected data. It supports Gremlin, Cypher, SPARQL, GQL, GoogleSQL, SQL:2023, and DQL across 35+ graph databases — all in one unified interface.

## What is gdotv?

gdotv (formerly G.V()) is a desktop and web-based graph database client built by [gdotv Ltd](https://gdotv.com), a Glasgow, Scotland-based software company. It gives graph database users — from beginners to experts — a powerful, unified toolset to accelerate their day-to-day graph project work.

Key capabilities:

- **The first-ever Gremlin query debugger** — traversal simulation and step-by-step inspection
- **Schema-aware autocomplete** for Gremlin, Cypher, SPARQL, GQL, and more
- **Real-time graph visualization** with WYSIWYG editing and WebGL rendering (tens of thousands of elements)
- **Multi-connection management** across LPG and RDF databases in one interface
- **Automatic graph data schema discovery** via database APIs or efficient sampling queries
- **Interactive graph dashboards** for visual analytics
- **SPARQL query guardrails** to catch errors before execution
- **No-code graph exploration** and node/edge editing
- **Geospatial map view**, property-based filters, fuzzy element search
- **AI-powered text-to-query generation** (optional OpenAI integration)
